# Consolas de Videojuegos

## Instalación de un juego

=== "PlayStation"

```text
    Biblioteca -> Juego -> Descargar
```

=== "Xbox"

```text
    Mis Juegos -> Instalar
```

=== "PC"

```bash
    steam install game
```

## Consolas populares

| Consola | Fabricante |
|---------|------------|
| PlayStation 5 | Sony |
| Xbox Series X | Microsoft |
| Nintendo Switch | Nintendo |

    El almacenamiento disponible puede limitar la instalación de juegos.
